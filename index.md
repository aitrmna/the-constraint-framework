<h1>The Constraint Framework</h1>

<p><strong>Core Principle:</strong> Strong hypotheses predict everything observed, predict nothing not observed.</p>

<div style="background: var(--bg-alt); padding: 1rem 1.25rem; border-radius: 8px; margin: 1.5rem 0; border: 1px solid var(--border);">
  <p style="margin: 0; font-weight: 600; color: var(--accent);">What makes this different</p>
  <p style="margin: 0.5rem 0 0 0; color: var(--text-muted); font-size: 0.95rem;">Most analysis asks "Is this possible?" This framework asks "What does this predict?" and ruthlessly eliminates hypotheses that fail their own predictions.</p>
</div>

<hr>

<h2>The Problem</h2>

<p>Traditional reasoning fails:</p>
<ul>
  <li><strong>Possibility bias:</strong> "It's possible" → stops there</li>
  <li><strong>Confirmation bias:</strong> Only looks for supporting evidence</li>
  <li><strong>Motivated reasoning:</strong> Reaches desired conclusion first</li>
  <li><strong>False equivalence:</strong> Treats all explanations as equally valid</li>
</ul>

<p><strong>What we need:</strong> Objective way to compare competing explanations.</p>

<hr>

<h2>The Solution</h2>

<p><strong>Every hypothesis, if true, predicts specific observable consequences.</strong></p>

<ol>
  <li>List what each hypothesis predicts</li>
  <li>Check predictions against reality</li>
  <li>Count confirmations vs failures</li>
  <li>Weight severity of failures</li>
  <li>Compare scores across hypotheses</li>
</ol>

<p><strong>Result:</strong> Objective, quantifiable comparison.</p>

<h2>When It Works</h2>

<p><strong>Best results when:</strong></p>
<ul>
  <li>Physical evidence exists</li>
  <li>Multiple independent witnesses</li>
  <li>Known comparison cases</li>
  <li>Checkable predictions</li>
</ul>

<p><strong>Struggles when:</strong></p>
<ul>
  <li>No physical evidence</li>
  <li>Single witness only</li>
  <li>Evidence destroyed/lost</li>
  <li>Purely philosophical questions</li>
  <li>Unfalsifiable claims</li>
</ul>
