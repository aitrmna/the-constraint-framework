<h2>The Framework</h2>

<p style="background: var(--bg-alt); padding: 1.25rem; border-radius: 6px; margin: 1.5rem 0;">
<strong>Core principle:</strong> True hypotheses constrain reality. False ones don't.<br><br>
If something is TRUE → it predicts what we observe AND doesn't predict what's absent<br>
If something is FALSE → reality contradicts it somewhere
</p>

<p><strong>Two questions for every claim:</strong></p>

<ol>
  <li><strong>"What MUST be true if this were ET/non-human?"</strong> → Check if present</li>
  <li><strong>"What would NEVER be true if this were mundane?"</strong> → Check if absent</li>
</ol>

<p>We score hypotheses numerically. Count confirmations vs failures. Weight severity. Compare ratios.</p>

<p><strong>Result:</strong> Objective ranking. No hand-waving. No "could be." Just: Does evidence survive testing or not?</p>

<h2>Why This Matters</h2>

<p>The question "Are we alone?" is answerable through evidence, not speculation.</p>

<p><strong>Bad approach:</strong> "UAPs could be aliens, could be secret tech, could be optical illusions... who knows?"</p>

<p><strong>Framework approach:</strong></p>
<ol>
  <li>List all explanations</li>
  <li>Generate risky predictions for each</li>
  <li>Check predictions against sensor data, official reports, physics</li>
  <li>Count failures</li>
  <li>Rank numerically</li>
</ol>

<p><strong>Outcome:</strong> USS Nimitz = 78% non-human (only hypothesis with zero critical failures). Not "maybe aliens." Quantified probability based on surviving constraints.</p>
