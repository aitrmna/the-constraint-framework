<h1>The Constraint Framework</h1>

<div style="background: var(--bg-alt); padding: 1.5rem; border-radius: 8px; margin: 2rem 0; border-left: 4px solid var(--accent);">
  <p style="margin: 0; font-size: 1.1rem; font-weight: 600;">The Nucleus</p>
  <p style="margin: 0.75rem 0 0 0; font-size: 1rem; line-height: 1.6;">True hypotheses constrain reality. False ones don't.</p>
  <p style="margin: 0.5rem 0 0 0; color: var(--text-muted); font-size: 0.95rem;">If something is TRUE → it predicts what we observe AND doesn't predict what's absent<br>
  If something is FALSE → reality contradicts it somewhere</p>
</div>

<h2>The Operational Form</h2>

<p>Ask two questions about every hypothesis:</p>

<ol>
  <li><strong>"What would we HAVE to see if this were true?"</strong> → Check if present</li>
  <li><strong>"What would we NEVER see if this were true?"</strong> → Check if absent</li>
</ol>

<p>Reality doesn't care about your hypothesis. If it's wrong, reality will contradict it somewhere. The more predictions you make, the more opportunities for reality to expose falsehood.</p>

<p><strong>True hypotheses survive repeated contact with reality. False hypotheses get cut.</strong></p>

<hr>

<h2>Why Traditional Reasoning Fails</h2>

<ul>
  <li><strong>Possibility bias:</strong> "It's possible" → stops there, never tests</li>
  <li><strong>Confirmation bias:</strong> Only looks for support, ignores contradictions</li>
  <li><strong>Motivated reasoning:</strong> Starts with conclusion, works backward</li>
  <li><strong>False equivalence:</strong> Treats all explanations as equally valid</li>
</ul>

<p>The framework forces you to <strong>risk being wrong</strong> — then checks if you are.</p>

<hr>

<h2>How It Works</h2>

<ol>
  <li>List all serious hypotheses</li>
  <li>For each: Generate risky predictions (things that could prove it false)</li>
  <li>Check predictions against reality</li>
  <li>Count confirmations vs failures</li>
  <li>Weight severity (critical failures vs minor ones)</li>
  <li>Compare scores numerically</li>
</ol>

<p><strong>Result:</strong> Objective ranking. Hypotheses that fail their predictions get eliminated. What survives is what fits reality.</p>

<h2>When It Works Best</h2>

<p><strong>Ideal conditions:</strong></p>
<ul>
  <li>Physical evidence exists</li>
  <li>Multiple independent witnesses</li>
  <li>Known comparison cases</li>
  <li>Checkable, risky predictions possible</li>
</ul>

<p><strong>Struggles with:</strong></p>
<ul>
  <li>No physical evidence</li>
  <li>Single witness only</li>
  <li>Evidence destroyed/unavailable</li>
  <li>Unfalsifiable claims</li>
  <li>Purely philosophical questions</li>
</ul>
